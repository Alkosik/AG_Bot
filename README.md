# AG_Bot
A sequel to the private Discord Bot Piesek.

## Including

### Commands

* #### Moderation
	* Ban,
	* Kick,
	* Mute,
	* Unban,
	* Unmute

* #### Fun
	* Roulette

* #### Utility
	* Disable Commands,
	* Emit,
	* Logtest

* #### APIs (depracted)
	* League of Legeds

* #### Other
	* Cipa,
	* Hot,
	* Ping

* Member Count
* Concert
